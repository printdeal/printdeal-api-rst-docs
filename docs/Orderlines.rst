Orderlines
==========

.. toctree::
   :hidden:

   Orderlines/Add_file

You can use regular HTTP methods to communicate with the API about orderlines:
    
- A ``POST`` request to ``/api/{orderlineId}/files`` will allow you to create a new file to an orderline.
    See :ref:`Adding file<Adding file>` for more information.
